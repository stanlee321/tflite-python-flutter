# TF Lite Python DEMO

This is a simple example of how to use TensorFlow Lite in Python.

## Requirements

```bash
pip install -r requirements.txt
```


## Usage

```bash
python main.py
```

Chnage your camera index in `main.py` if you have multiple cameras.

# Open a handle to the default webcam
